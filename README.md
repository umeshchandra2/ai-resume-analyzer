# AI Resume Analyzer

A Full Stack AI Resume Analyzer built using FastAPI and React.

## Features

- Upload Resume PDF
- Extract Skills from Resume
- Compare Resume with Job Description
- ATS Score Calculation
- Missing Skills Detection
- Resume Improvement Suggestions

## Tech Stack

### Frontend
- React
- Axios
- CSS

### Backend
- FastAPI
- PyPDF
- Python

## Run Locally

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
